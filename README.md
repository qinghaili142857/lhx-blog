## 食用方法(以Netlify举例)
1.  [Fork本项目](https://github.com/yCENzh/fuwari/fork)
2.  使用 [Cloudflare](https://netlify.com/) 创建一个项目 -> 导入现有Git存储库 -> 保存并构建(本项目以pnpm构建)
3.  没了,非常的简单,后续食用方法请查看我的博客
